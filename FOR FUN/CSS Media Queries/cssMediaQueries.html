what is Media Queries

  • Media Queries allows you to set different rules for different devices through simple queries about the device's features, such as screen width, device's width, color and orientation
  • Media query is a CSS technique introduced in CSS3.
  • It uses the @media rule to include a block of CSS properties only if a certain condition is true.
  • Breakpoint - for the window size

How to create a query

CSS Jump Start: 04 Media Queries

  • Media Type

    • All-all screens
    • Print - page for printing
    • Screen - computer screen
    • Speedch - speech reader
    • Braille - braille printer
    • Embossed
    • Handheld - handhelp devices
    • Projection - projection tvs
    • TTY
    • TV

  • Media Features

    • Width/Height
    • Device-Width/Device-Height
    • Orientation - Landscape/Portrait View cellphone rotation(table/colom)
    • Aspect-ratio - device aspect ratio is 16:9 or 16:10 for example
    • Device-aspeact-ratio
    • Color
    • Color-index
    • Monochrome
    • Resolution
    • Scan
    • Grid

  • Logical Operators

    • And - if all conditions are true
    • Or - comma separated, either one condition is true
    • Not - only is this condition is not true
      • @media(min-width: 500px){
        support page width less than 500px
        }
      • @media(min-width: 500px) and (max-width: 1000px){
        support page width between 500px to 1000px
        }
      • @media screen and not (device-aspect-ratio:4/3){
        support computer screen but not in ratio 4:3
        }
      • @media screen and (device-aspect-ratio:16/9), screen and (device-aspect-ratio: 16/10){
        support computer screen but not in ratio 16:10
        }
