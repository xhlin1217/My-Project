Authentication Server

Create a web application that securely authenticates people with a username and password.

Minimum Requirements

  • Back-End MongoDB support
  • Create JWT (JSON WEB TOKEN) when user login, and auto pick up JWT when loading or refresh page
  • Destroy JWT when user logout
  • Date validate check on front-end and back-end
  • User email confirmation after account register and activate account
